# 🎮 快速开始

## 3 分钟安装指南

### 第一步：复制数据包 (30 秒)
1. 找到 `lucky_block_datapack` 文件夹
2. 复制到你的世界存档的 `datapacks` 文件夹
   - Windows: `%appdata%\.minecraft\saves\你的世界名\datapacks`
   - Mac: `~/Library/Application Support/minecraft/saves/你的世界名/datapacks`
   - Linux: `~/.minecraft/saves/你的世界名/datapacks`

### 第二步：加载数据包 (30 秒)
1. 进入 Minecraft 世界
2. 按 `T` 打开聊天
3. 输入 `/reload`
4. 看到绿色消息表示成功

### 第三步：获得幸运方块 (10 秒)
```
/function lucky_block:give_lucky_block
```

### 第四步：玩耍！ (2 分钟)
1. 放置金块
2. 打破它
3. 享受惊喜！✨

## 🎯 核心功能

- ✨ **90% 几率** - 等级 50 随机宝可梦
- 🌟 **10% 几率** - 等级 50 闪光宝可梦
- 🎆 华丽的粒子和声音效果
- 📝 精美的聊天消息

## ⚡ 常用命令

| 命令 | 说明 |
|------|------|
| `/function lucky_block:give_lucky_block` | 获得幸运方块 |
| `/reload` | 重新加载数据包 |
| `/datapack list` | 查看数据包列表 |

## 🔧 遇到问题？

### 数据包没加载？
```
1. 检查文件夹位置是否正确
2. 确保 pack.mcmeta 文件存在
3. 使用 /datapack list 查看
```

### 宝可梦不生成？
```
1. 确保安装了 Cobblemon 模组
2. 测试 /pokespawn random level=50 命令
3. 检查是否有足够空间
```
