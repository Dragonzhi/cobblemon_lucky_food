# Cobblemon 幸运食物数据包 - 快速指南

## 3 分钟开始

### 1. 安装
将 `cobblemon_lucky_food_datapack` 文件夹复制到 `%appdata%\.minecraft\saves\你的世界名\datapacks`

### 2. 加载
```
/reload
```

### 3. 获得幸运食物

| 食物 | 命令 |
|------|------|
| 普通 | `/function cobblemon_lucky_food:give_lucky_food` |
| 传说 | `/function cobblemon_lucky_food:give_legendary_lucky_food` |
| 全图鉴 | `/function cobblemon_lucky_food:give_weighted_lucky_food` |

### 4. 使用
手持幸运食物（金色名称）→ 右键吃完 → 享受惊喜！

## 功能

| 效果 | 几率 | 说明 |
|------|------|------|
| 普通宝可梦 | 90% | 等级 50 随机宝可梦 |
| 闪光宝可梦 | 10% | 等级 50 闪光宝可梦 |
| 传说宝可梦 | - | 94只传说/幻之宝可梦随机 |
| 全图鉴模式 | - | 每只不重复,全收集后提示重置 |

## 常用命令

```
/function cobblemon_lucky_food:give_lucky_food      - 获得普通幸运食物
/function cobblemon_lucky_food:give_legendary_lucky_food - 获得传说幸运食物
/function cobblemon_lucky_food:give_weighted_lucky_food  - 获得全图鉴幸运食物
/function cobblemon_lucky_food:debug_lucky_food     - 诊断状态
/function cobblemon_lucky_food:reset_all_lucky_food  - 一键重置
/function cobblemon_lucky_food:check_weight_lucky_food   - 全图鉴收集进度
/function cobblemon_lucky_food:reset_weight_lucky_food   - 重置全图鉴进度
/reload                                              - 重新加载
/datapack list                                       - 查看数据包
```
