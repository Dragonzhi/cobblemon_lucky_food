# 🎮 Cobblemon 幸运方块数据包 - 快速指南

## 🚀 3 分钟开始

### 1️⃣ 安装
将 `lucky_block_datapack` 文件夹复制到：
```
%appdata%\.minecraft\saves\你的世界名\datapacks
```

### 2️⃣ 加载
进入游戏，按 T 输入：
```
/reload
```
看到绿色消息表示成功！

### 3️⃣ 获得幸运方块
```
/function lucky_block:give_lucky_block
```

### 4️⃣ 使用
1. 放置幸运方块（金色名称）
2. 打破它
3. 享受惊喜！✨

## ✨ 功能

| 效果 | 几率 | 说明 |
|------|------|------|
| 🐾 普通宝可梦 | 90% | 等级 50 随机宝可梦 |
| ✨ 闪光宝可梦 | 10% | 等级 50 闪光宝可梦 |
| 🎆 粒子效果 | 100% | 华丽特效 |
| 🔊 声音 | 100% | 悦耳音效 |
| 📝 聊天消息 | 100% | 精美提示 |

## ⚠️ 重要

- ✅ **只有幸运方块会触发**（金色名称）
- ❌ **普通金块不会触发**
- ✅ **不修改原版内容**
- ✅ **可重复使用**

## 🎯 常用命令

```
/function lucky_block:give_lucky_block  - 获得幸运方块
/rereload                                - 重新加载
/datapack list                           - 查看数据包
```
