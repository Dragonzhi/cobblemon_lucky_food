## 🎯 核心文件说明

### 1. `break_lucky_block.json` (进度触发器)
```
作用：检测玩家破坏金块的行为
触发：玩家破坏任意金块
奖励：执行 lucky_block:on_break 函数
```

### 2. `on_break.mcfunction` (主函数)
```
作用：处理幸运方块破坏事件
执行：
  1. 播放 3 种粒子效果
  2. 播放声音
  3. 随机生成 0-9 的数字
  4. 0 = 闪光宝可梦，1-9 = 普通宝可梦
  5. 重置计分板
```

### 3. `spawn_normal.mcfunction` (普通生成)
```
作用：生成普通宝可梦
命令：/pokespawn random level=50
消息：金色加粗提示
```

### 4. `spawn_shiny.mcfunction` (闪光生成)
```
作用：生成闪光宝可梦
命令：/pokespawn random level=50 shiny
消息：彩色渐变提示
特效：额外 100 个发光粒子
声音：末影龙成长声
```

### 5. `give_lucky_block.mcfunction` (给予物品)
```
作用：给予玩家幸运方块
物品：金块
NBT: 
  - CustomName: "幸运方块" (金色加粗)
  - LuckyBlock: 1b (标记)
  - Lore: 描述文字
```

### 6. `init.mcfunction` (初始化)
```
作用：数据包加载时执行
功能:
  - 创建 lucky_shiny 计分板
  - 显示欢迎消息和使用说明
```
