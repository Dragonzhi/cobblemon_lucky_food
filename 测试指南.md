# 幸运方块数据包 - 快速测试指南

## 🧪 测试步骤

### 1. 安装数据包
```
1. 将 lucky_block_datapack 文件夹放入世界存档的 datapacks 文件夹
2. 进入游戏，按 T 打开聊天栏
3. 输入 /reload
4. 应该看到：[Cobblemon 幸运方块] 数据包已加载!
```

### 2. 获得幸运方块
```
/function lucky_block:give_lucky_block
```
你应该获得一个名为"幸运方块"的金色金块

### 3. 测试功能
```
1. 放置幸运方块
2. 打破它
3. 观察效果:
   - ✨ 粒子效果
   - 🔊 声音效果
   - 📝 聊天消息
   - 🐾 宝可梦生成
```

### 4. 多次测试 (测试闪光几率)
```
重复以下命令多次:
/function lucky_block:give_lucky_block

然后放置并打破，统计闪光宝可梦出现次数
10 次中应该有 1 次左右是闪光
```

## ✅ 预期结果

### 普通宝可梦 (90% 几率)
- 粒子效果：快乐村民、发光、附魔击中
- 声音：经验球拾取声
- 消息：✨ 幸运方块生成了一只野生宝可梦！ (金色加粗)
- 生成：等级 50 的随机宝可梦

### 闪光宝可梦 (10% 几率)
- 粒子效果：上述效果 + 额外大量发光粒子
- 声音：经验球拾取声 + 末影龙成长声
- 消息：✨ 幸运方块生成了一只闪光宝可梦！ (彩色)
- 生成：等级 50 的闪光随机宝可梦

## 🐛 问题排查

### 如果看不到加载消息
```
/datapack list
# 检查 lucky_block_datapack 是否在列表中
```

### 如果命令不工作
```
# 检查 Cobblemon 是否安装
/pokespawn random level=50
# 这个命令应该能正常执行
```

### 如果进度不触发
```
# 重置进度
/advancement revoke @s lucky_block:break_lucky_block
# 然后重新打破金块
```
